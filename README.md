# Presidential-Election-Analysis

Authors: Kashvi Chandwani and Kasey Tian

This is the code for the project for Cloud Computing. THe goal of this project is to provide visualizations of each presidential election to help us see party dominance in the USA over the years. 

FILE SUMMARY: 
extractdata.py -- extract the data from the website
cleandata.py -- cleaned and reformatted the data

jupyter/make_graph.ipynb -- process the data using pySpark
